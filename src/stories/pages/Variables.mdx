Here is a list of all available variables and their values.

- [Colours](#colours)

## Colours

```Sass
// Main

$white: #FFFFFF;
$black: #000A1E;

// Greys

$grey-90: #1A2335;
$grey-80: #333B4B;
$grey-70: #4D5462;
$grey-60: #666C78;
$grey-50: #7F848E;
$grey-40: #999DA5;
$grey-30: #B2B5BB;
$grey-20: #CCCED2;
$grey-10: #E5E6E8;
$grey-5: #F2F3F4;

// Background

$background-default: $black;
$background-paper: $grey-90;

// Text

$text: $white;
$text-contrast: $black;

// Brand

$primary: #00F3FF;
$primary-contrast: $text-contrast;

$secondary: #E7FC00;
$secondary-contrast: $text-contrast;

$highlight: #FF74E6;
$highlight-contrast: $text-contrast;

// Feedback

$success: #27AE60;
$success-contrast: $text;

$info: #17A0CC;
$info-contrast: $text;

$warning: #FFDC46;
$warning-contrast: $text-contrast;

$error: #EB5757;
$error-contrast: $text;

// Others

$border-color: rgba($text, 0.2);
$border: 1px solid $border-color;
$border-color-contrast: rgba($text-contrast, 0.2);
$border-contrast: 1px solid $border-color;

$hover: rgba($text, 0.12);
$hover-contrast: rgba($text-contrast, 0.12);

$overlay: rgba($text-contrast, 0.8);
$overlay-contrast: rgba($text, 0.2);
```
