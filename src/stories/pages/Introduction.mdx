# Pallote

Pallote is a design system and a React UI component library. Here you will find our design guidelines, component documentation and resources on how to use and customise the library.

## Install

```shell
npm install --save-dev pallote-react
```

## Dependencies

Pallote uses the `pallote-css` npm package, a simple and stylish CSS component library, as it's styling library.

For more information on the project, go to the [documentation](https://pallote.com/) website.
